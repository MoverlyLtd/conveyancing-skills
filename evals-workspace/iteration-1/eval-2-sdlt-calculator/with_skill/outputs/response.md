# Stamp Duty on a Second Property at £650,000

**Total SDLT: £55,000** (effective rate: 8.46%)

Because this is an additional (second) property, the 5% higher-rate surcharge applies on top of the standard residential rates. Here's the band-by-band breakdown:

| Band | Rate | Tax |
|------|------|-----|
| £0 – £125,000 | 5% | £6,250 |
| £125,001 – £250,000 | 7% | £8,750 |
| £250,001 – £650,000 | 10% | £40,000 |
| **Total** | | **£55,000** |

### Key things to note

- **Additional property surcharge:** The 5% surcharge is added to each band because you already own another property. For example, the first £125,000 is normally 0% but becomes 5% with the surcharge.
- **Filing deadline:** You must file your SDLT return and pay the tax within **14 days of completion**.
- **Replacement exception:** If you're selling your current main residence and replacing it with this property, you may be able to reclaim the 5% surcharge (you have 36 months to sell the old property and apply for a refund).

This applies to purchases in **England and Northern Ireland** only. Scotland uses LBTT and Wales uses LTT, which have different rates.
