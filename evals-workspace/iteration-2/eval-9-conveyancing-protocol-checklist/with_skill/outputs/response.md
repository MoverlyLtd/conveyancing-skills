# Pre-Exchange Checklist — Law Society Conveyancing Protocol

You're at the critical point just before exchange. Here's what should be checked and confirmed against the Protocol, broken down by role.

> **Note:** This is compliance guidance based on the Law Society Conveyancing Protocol (2019 edition). Always refer to the current published protocol for authoritative requirements. The client's best interests override protocol obligations in all cases.

---

## General Obligations to Confirm Before Exchange

These apply throughout the transaction but should be verified as complete before you exchange:

| # | Check | Buyer | Seller | Ref |
|---|-------|:-----:|:------:|-----|
| G1 | Have you verified the identity of the other side's conveyancer? | ✓ | ✓ | General, para (a) |
| G2 | Is the contract using the latest edition of the Standard Conditions of Sale? | ✓ | ✓ | General, para (b) |
| G3 | Is the Code for Completion by Post incorporated (this is automatic under the protocol)? | ✓ | ✓ | General, para (c) |
| G4 | Are all property forms the current editions (TA6, TA7, TA10 etc.)? | ✓ | ✓ | General, para (d) |
| G5 | Have you agreed which formula you'll use for telephone exchange? | ✓ | ✓ | General, para (e) |
| G6 | Are any amendments to Standard Conditions limited to essential changes only? | ✓ | ✓ | General, para (f) |
| G7 | If varying the Code for Completion by Post, do you have specific client instructions for this? | ✓ | ✓ | General, para (g) |
| G8 | Were any additional enquiries raised limited to those relevant to and necessary for this transaction? | ✓ | — | General, para (l) |

## Key Things to Double-Check Right Before Exchanging

Based on the protocol requirements, these are the items most commonly missed or problematic at exchange:

1. **Search currency** — Are your local authority, drainage, and environmental searches still within their validity period? (Stage B, para 11)
2. **Mortgage conditions** — Have all conditions in the mortgage offer been satisfied? Any special conditions that could delay? (Stage B, para 12)
3. **Deposit availability** — Is the deposit cleared and ready to send? If part of a chain, is the deposit arrangement agreed? (Stage B, para 9)
4. **Outstanding enquiries** — Have all raised enquiries been answered to your satisfaction? (Stage B, paras 3 & 6)
5. **TA10 agreement** — Has the fixtures and fittings list been finalised and agreed by both parties? Disputes after exchange are problematic. (Stage B, para 10)
6. **Client authority** — Do you have your client's explicit authority to exchange on the agreed terms and completion date? (Stage B, para 8)
7. **Buildings insurance** — Is the buyer ready to insure from exchange? The risk passes at that point. (Stage C, para 8)
