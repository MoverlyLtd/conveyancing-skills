For boundary ownership, you need:

`/propertyPack/legalBoundaries/ownership/uniformBoundaries/`

This has four sub-paths for each side:
- `.../left` — left boundary owner
- `.../right` — right boundary owner  
- `.../rear` — rear boundary owner
- `.../front` — front boundary owner

Each accepts one of: `Seller`, `Shared`, `Neighbour`, or `Not known`.

**Note:** This is only for uniform boundaries (where each side has a single owner). If boundaries are mixed ownership, use `/propertyPack/legalBoundaries/ownership/attachments` to upload a boundary plan instead.
