The PDTF schema has a path for recording non-residential use of the property:

/propertyPack/alterationsAndChanges/nonResidentialPurposes

This path would be used to record that part of the property is used as a home office, rather than solely for residential purposes.

The key fields under this path are:

- `/propertyPack/alterationsAndChanges/nonResidentialPurposes/usedForNonResidentialPurposes`: A yes/no flag indicating if any part of the property is used for non-residential purposes.
- `/propertyPack/alterationsAndChanges/nonResidentialPurposes/nonResidentialUseDescription`: A description of the non-residential use, e.g. "Home office".
- `/propertyPack/alterationsAndChanges/nonResidentialPurposes/attachments`: Any supporting documents related to the non-residential use, such as planning permissions or lease agreements.

So in summary, the correct PDTF path to record a home office use would be:

/propertyPack/alterationsAndChanges/nonResidentialPurposes
